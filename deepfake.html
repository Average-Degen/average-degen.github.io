Degenerate AI Guides

This guide will being using a fork of a popular one-click face swap tool called Roop
We will be using a fork called Roop Unleashed. This is because Roop has been discontinued and Roop Unleashed has more features

Installation Guide

The easiest way to install Roop Unleashed is by using the one-click installer

Step 1: Download Installer

This installer can be found in the releases page of the GitHub repository
You just need to download the latest roop-unleashed_win zip file and place it in the directory that you want to install to

Step 2: Extract Contents

Now to install the program you need to first extract the zip file
To do this you can either use a program like WinRAR or you can just right click the file and select Extract All

Step 3: Run Roop Unleashed

Now that you have extracted the contents of the zip file you can find the file windows_run.bat you use this file to launch the web UI
The first run will take some time because it has to download all the required files

Once the setup is completed it will open to the address 127.0.0.1:7860
This means that you can't open both Stable Diffusion and Roop Unleashed at the same time without changing the port number (7860) in either of the program's settings
I recommend changing the port of Stable Diffusion as I have had some errors trying to modify the settings in Roop Unleashed
To change the port you simply add --port xxxxto the commandline args of webui-user.bat and replace xxxx with your desired port number

Side Note

If you encounter any errors you may need to download and install Visual Studio and select the C++ package during installation

Deepfaking

Inputs

Choosing Input

When choosing an input for a face you are looking for an image that is not blurry and has good lighting

When choosing a target look for something that features an unobstructed face that is facing the camera

Inputting

Before you do anything you first need to upload the inputs for deepfaking

The first upload box titled Source Face Image is where you will upload the images of the face that you want to transplant onto the deepfake

Once you have uploaded an image here it will be processed and the image of the face will appear in the box above
This is where you will select the face that you want to use if the image contains multiple faces

The Target File(s) box is where you can upload as many destination files as you want
Roop Unleashed supports deepfaking of both images and videos

Settings

Face Swap Method

Here is where you select the method that will be used to find the faces to target when swapping

All the options are quite self-explanatory, you just choose which one best suits your desired result

The Max Face Similarity Threshold defines how strict to be when checking for the face selected by the swapping method
This does not apply if using All Faces swapping method

Post-Processing

Using post-processing will drastically slow down Roop Unleashed but will produce higher quality results

This is nessesary for high quality results as the developer of the model used by Roop Unleashed has only realased a 128x128 resolution version
This means that the maximum resolution for output faces is 128x128 pixels without post-processing

The upscaler that you use depends on your personal preference but I find that DMDNetpreserves facial features the best

The Original/Enhanced image blend ratio defines how much to blend the upscaled face with the original
This is another setting you should play around with to find the best result

Generation

To deepfake all the files that you uploaded all you need to do is press the orange Start button and wait for it to finish

Once done the results will appear in the roop-unleashed/output folder
I recommend going to the settings tab and unchecking the option that says Clear output folder before each run as this will delete all previous results from the output folder and may result in lost files

If you want to upload any videos you do to 4chan you can also change the Video Output Format to webm and decrease the Video Quality (crf) to decrease file size

Make sure you press Apply Settings after changing any of these values
